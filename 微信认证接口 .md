


#微信企业号认证接口

##请求说明
###请求方式: 
POST

dataType:"json"

Content-Type: application/json

###URL:
http://ghxs.88u.cas.scut.edu.cn/Wisdom/wx/getAuthentication

###请求包结构体为:
```
{
"code":在企业号里面打开页面微信会提供一个code(String)
}

```

---


###返回结果
成功时：
```
{
" account":用户的中央认证帐号 (String)
" result":"success"(发送成功)
}
```
失败时：
```
{

"result":"fail"(code错误)

}
```
